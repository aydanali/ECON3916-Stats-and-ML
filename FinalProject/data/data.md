# Dataset

## CO2 and Greenhouse Gas Emissions

This data was provided by **Our World in Data's CO2 and Greenhouse Gas Emissions**. 

This the premeire dataset for country emission data used by the World Bank and many others as it is updated regularly.
Given its size, I cannot upload the full CSV used (github's 25MB limit for web uploads). Below is the github link where I got the raw link to place in my code. 
Please find the updated and accurate version there as well as documentation. 

https://github.com/owid/co2-data/tree/master

*For reproducability I have given the cleaned dataset in the data folder.*

## From the World Bank (https://data360.worldbank.org/en/dataset/OWID_CB):

This CO2 and Greenhouse Gas Emissions dataset is a collection of key metrics maintained by Our World in Data. It is updated regularly and includes data on CO2 emissions (annual, per capita, cumulative and consumption-based), other greenhouse gases, energy mix, and other relevant metrics.

For further details, please refer to https://github.com/owid/co2-data
